## Bean and Brew ☕️
A mock storefront and shopping cart for a coffee bean retailer; created as a project for The Odin Project.

### Framework & Concepts used:
* Next.js & React
* Server-side rendering & client-side rendering
* Context
* Testing and mocking with Jest
